Diese Applikation zur Entspannung ist wie folgt zu handhaben:
  
  beenden der Applikation -> ESC, q, Q
  Anzahl der Elemente aendern -> Pfeiltaste hoch oder runter
  Bewegung der Elemente um die y-Achse -> Pfeiltaste links oder rechts
  Bewegung der Elemente durch Maus bei gedrueckter linker Maustaste um alle Achsen
  eigene Musikdatei im .wav Format abspielen -> F1
  Musik ein- oder ausschalten -> s, S
  Kugeln durch Teekanne ersetzen -> t, T
